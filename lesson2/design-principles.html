PARC: Alignment

Nike

nike.com

Nike's website has a flush-left alignment which unifies and connects each element well. There is an invisible line from the top left of the website through the bottom left of the website. I drew the blue line so that it is easy to see what the "invisible line" is. This straight line gives the website strength and sophistication.

PARC: Repetition

Loui Vuitton

louisvuitton.com

Throughout the website, it uses the same exact font, font-size, and the size of each picture. The principle of repetition helps organize the information so that a user can easily navigate the website. It looks well balanced, organized, and clean. Those are good aspects of a professional web design.

PARC: Contrast

Betsey Johnson

betseyjohnson.com

With the subtle light blue background, hot pink makes a strong contrast which catches a user's eyes. Even though the font size is smaller, I was interested in "SHOP BETSEY'S FAVES" more than the heading. Also, the colorful colored bag becomes a strong accent piece that balances out more subtle looking accessories. The contrast among the accessories organized the website.
